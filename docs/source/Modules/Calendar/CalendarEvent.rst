CalendarEvent
=============

.. js:attribute:: calendarevent.id

    ``number``. The unique ID for this activity.

.. js:attribute:: calendarevent.

    ``number``. The type of activity.type

.. js:attribute:: calendarevent.title

    ``String``. The name of the activity.

.. js:attribute:: calendarevent.extendedTitle

    ``String``. The full title of the activity as seen on Compass. States the start time, title, location, and manager.

.. js:attribute:: calendarevent.description

    ``String``. A description of the event.

.. js:attribute:: calendarevent.start

    ``Date``. The start time of the event.

.. js:attribute:: calendarevent.finish

    ``Date``. The finish time of the event.

.. js:attribute:: calendarevent.allDay

    ``bool``. Whether the event lasts the whole day.

.. js:attribute:: calendarevent.cancelled

    ``bool``. Whether the event has been cancelled.

.. js:attribute:: calendarevent.rollMarked

    ``bool``. Whether the roll has been marked for the event.
